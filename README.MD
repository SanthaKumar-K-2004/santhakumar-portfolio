## 🚀 Overview

A high-performance, interactive 3D portfolio website showcasing my journey as a Full-Stack Developer and AI Integration Specialist. Built with a focus on immersive user experience and optimized performance.

## ✨ Key Features

- **🎮 Interactive 3D Models**: Using Three.js and React Three Fiber for a premium desktop and globe experience.
- **⚡ Smart WebGL Optimization**: Custom lazy-loading and error handling to prevent GPU context loss and ensure smooth performance across all devices.
- **📱 Fully Responsive**: Seamless experience from high-end desktops to mobile devices.
- **🎭 Dynamic Animations**: Polished transitions powered by Framer Motion.
- **📧 Integrated Contact System**: Streamlined communication using EmailJS with custom branding.

## 🛠️ Tech Stack

- **Frontend**: React.js, Vite
- **3D Graphics**: Three.js, React Three Fiber, React Three Drei
- **Styling**: Tailwind CSS, Vanilla CSS
- **Animation**: Framer Motion
- **Icons & Graphics**: FontAwesome, Custom SVGs

## 📁 Project Structure

```text
src/
├── assets/         # Optimized visual assets & logos
├── components/     # Reusable React components & 3D Canvases
├── constants/      # Centralized project & experience data
├── hoc/            # Higher-Order Components for layout
├── utils/          # Motion configurations & utilities
└── App.jsx         # Root application structure
```

## 🛠️ Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/SanthaKumar-K-2004/santhakumar-portfolio.git
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Configure Environment Variables**
   Create a `.env` file in the root directory:
   ```env
   VITE_EMAILJS_SERVICE_ID=your_id
   VITE_EMAILJS_TEMPLATE_ID=your_id
   VITE_EMAIL_JS_ACCESS_TOKEN=your_token
   ```

4. **Launch Development Server**
   ```bash
   npm run dev
   ```
